# Responsive Gym Website 💪

A modern and responsive **Gym/Fitness Website** built using **HTML, CSS, and JavaScript**.
This project was created as part of a **web development internship project** to demonstrate front-end development skills.

---

## 📌 Features

* Responsive navigation menu
* Modern and clean UI design
* Fitness program showcase
* Pricing plans section
* BMI (Body Mass Index) calculator
* Email subscription form
* Scroll-up button
* Smooth scrolling navigation
* Mobile-friendly responsive layout

---

## 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and responsive design
* **JavaScript** – Interactive features
* **Google Fonts** – Typography
* **Remix Icons** – Icons
* **EmailJS** – Email functionality
